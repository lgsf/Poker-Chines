Informações


O poker Chinês é um jogo de cartas não muito conhecido. Apesar de usar as mãos do Poker tradicional, o Poker Chinês é um jogo com uma dinâmica diferente.

Dois, três ou quatro jogadores podem jogar. Cada jogador recebe mãos de 13 cartas. O intuito é que o jogador descarte todas suas cartas primeiro, vencendo assim a rodada. Após um número de rodadas, uma partida é contabilizada e um campeão é coroado.

O problema nasce aí: a cada rodada é necessário que o número de cartas de cada jogador seja contabilizada. Com o número de rodadas crescente, esse processo se torna cansativo.

Surgiu assim um dia a ideia de automatizar tal processo com a criação de uma página que seria usada para contabilizar tais pardidas. Outro problema foi resolvido por consequência, agora seria possível manter o histórico das partidas.
